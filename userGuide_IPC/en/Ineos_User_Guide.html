Information Tabs

The tabs at the top right-hand side of the screen allow different information to be viewed about the current topic.

The active tab is highlighted dark blue.

Parts Tab

The Parts Tab shows a table of Information relevant to the current area of the vehicle being viewed:

    1.

    Item Numbers for 2D Drawing

    2.

    Service Part Number

    3.

    Service Description

    4.

    Quantity

    5.

    Remarks

Clicking on any row of the table will highlight that part on the 3D Model.

If no 3D model is available, but a 2D image is, then clicking the table row will open the 2D image.

Moving the mouse point over a table row without clicking on it will highlight it in blue.

The table row will change to a sand colour and remain this colour when selected by clicking on it.

Clicking on a Part on the 3D Model will highlight the corresponding row on the table.

Selecting the small arrow next to each table row will open a small sub table of information relevant to that part. This includes:

    1.

    Order Quantity – Changed by the user to meet the required number

At the bottom of this sub-table is the Add to Order button. Clicking this adds the selected order quantity to the user’s basket.

The table will show Interchangeability information, but only when there is information available. If there is no interchangeability information, this section will not be displayed

Details Tab

The Details Tab shows a table of applicable vehicle, market and document information, to the current topic being viewed.

    1.

    Title of the document.

    2.

    Applicable section of the vehicle. I.e. powertrain, body, electrical.

    3.

    Reference number of the document.

    4.

    Vehicle model.

    5.

    Model year or range.

    6.

    Engine variant.

    7.

    Transmission variant.

    8.

    If the document relates to Left Hand Drive (LHD) or Right Hand Drive (RHD) variant.

    9.

    Applicable body type variant. I.e. LWB SW, XLWB PU.

    10.

    Applicable region.

    11.

    First applicable VIN

    12.

    Last applicable VIN

    13.

    Current version, along with date and reason for previous document revisions.

3D Model Control

The 3D Model can be controlled using the mouse.

The Model can be rotated freely 360° by clicking and holding the left-hand mouse button and moving the mouse to get the desired view.

Using the scroll wheel on the mouse allows the view to be zoomed in or out.

Clicking and holding the scroll wheel allows the model to be dragged.

Clicking on a part will highlight that part in a different colour.

Three Lines Icon

Selecting this icon opens a menu with the different view options for the selected topic. These are:

    >

    Location View

    >

    Assembled View

    >

    Exploded View

    >

    2D View

Location View, Assembled View and Exploded View apply only to 3D models. 2D View will be displayed only when there is no 3D model and will not be clickable.

3D models will always have an Assembled View. The topic may only have this one view, or it may have an Assembled and an Exploded View, an Assembled and a Location View, or all three views.

Location View

The Location View shows parts for the selected topic in the position they are used.

Assembled View

The Assembled View shows a close up view of the part or assembly in the selected topic.

Some larger part Assemblies are made up of smaller Sub-Assemblies. There are also Assembled Views for each of theses Sub-Assemblies.

Selecting an Assembled View for a Sub-Assembly also filters the Parts Table from all available parts for the section, to just the components of that Sub-Assembly.

Exploded View

Where applicable, the exploded view gives a clearer perspective of the parts within an assembly or sub-assembly.

Model Viewpoint

A preset view can be selected using the solid cube icon on the upper left-hand side of the 3D Model Pane.

Clicking the icon opens a menu to show seven different selectable views. The icon turns white and the menu background dark blue once opened.

When a view is selected, the icon for that view stays highlighted in white.

Six buttons have a cube outline with one solid face. The solid face represents the view the model will be put into. I.e. top, rear, left-hand side.

The last button shows three solid faces. This will put this model into an angled perspective using the shared corner as the center point.

Once opened, the menu remains open until the main icon is clicked again which then closes the menu.

The icon changes back to dark blue on a white background.

Model Tools

The cube outline icon on the upper left-hand side of the 3D Model Pane, can be used to change certain parameters relating to the 3D Model.

Clicking the icon turns it white on a dark blue background and opens a menu with four options.

The menu remains open until the main icon is clicked again to close the menu.

Set Centre

The first icon is Set Centre. Click on the icon and then click on the part or area you wish to centre. This then sets that as the centre to transform the model around.

Show Horizon Centre

The second icon is Show/Hide Horizon Center. Clicking this turns on a set of axes to show where the horizon center is currently set. Clicking it again hides the axes.

Full Screen- Remove Task Bars

The third icon is Full Screen. Clicking this removes the task bar and upper menus and fills the space with the topic page.

To exit Full Screen either click the icon again or press the ESC button on the keyboard.

Align Horizon

The fourth icon is Align Horizon. Clicking this aligns the 3D model to the horizon.

Full Screen - 3D Model

The white table on a dark blue background icon removes the right-hand text pane from view.

This makes the left-hand 3D Model pane full screen. The Icon changes to dark blue on a white background when this is active.

Pressing the icon again brings the text pane back into view. The icon changes back to the original colours.

Part Visibility

The Eye Icon and Eye with a Line through are used to turn part visibility on and off.

Click on the part on the model you wish to change. Once this is highlighted, clicking the Eye with a Line through will turn that part visibility to off. The part will remain off until it is either turned back on using the icon, or the animation changes the transparency.

To get the part back, whilst that part is still selected, click on the Eye Icon which will turn the visibility back on.

Transparency Slider

Select the part you want to change the transparency of then click this icon.

A slider bar will appear which can be moved in two ways.

One way is to click and hold the slider control and slide this along the bar to the required point.

The second way is to put the mouse pointer at a point on the bar and click. The slider control will jump to this point.

With the slider control fully to the left, objects will be fully opaque.

With the slider control fully to the right, objects will be fully transparent.

If changing the transparency of multiple items, close the transparency slider before selecting another item.

This ensures the slider control is in the correct position for both the new item and when returning to a previously changed item.

Show 2D

Clicking on the Scene Icon changes the 3D Colour Model to a 2D Line drawing.

If no 2D Drawing is available, then this Icon will be greyed out and no longer function.

Each Part is given a call out to identify it against the relevant line in the parts table.

Clicking on a call out will highlight it on the drawing and highlight the corresponding line of the parts table.

Clicking on a line on the parts table will highlight the corresponding call out on the model.

In 2D view, if there are multiple instances of a part, only one will be shown. All instances can be seen in the 3D view though.

Clicking the Scene Icon again changes the image back to the 3D Colour Model.

Reset View

The two Arrows in a circle is the Reset View tool.

Clicking on this icon resets the current to view to focus on the main content of the topic. This can be a single item, or an assembly.

Print Preview

Clicking the icon shaped like a printer will display the default print preview for your system.

Right Click Menu

With the mouse pointer over a part in the 3D model, or a table row on the parts table, clicking the right mouse button brings up a menu.

This menu contains a selection of functions for quick access. These are:

    1.

    Hide - Hides the part that the mouse pointer was over when the right mouse button was clicked.

    2.

    Isolate - Hides all parts except for the one that the mouse pointer was over when the right mouse button was clicked

    3.

    Show All - Shows all previously hidden parts.

    4.

    Copy - Copies the part number of the 3D part or parts table row the mouse pointer was over, ready for pasting.

    5.

    Pages - The list of pages from the three lines icon menu is displayed. The user can select any one of these to quickly change the displayed page. These options will change based on which pages are available to view in the current topic/catalogue area.

The right click menu will only display if the mouse pointer is over the 3D model. It will not display if the mouse pointer is over blank space or if the 2D view is selected.
